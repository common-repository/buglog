== Description ==

Bug Reporting Tool for Websites.

Buglog gives you clear and actionable feedback. Share bug reports with your team in real-time.

- Customize feedback widget to match your needs. Change color, language and more.
- Works Anywhere. On desktops, tablets, mobiles and supports all major browsers.
- Website users can report an issue via our feedback form with just one button click.
- Select a specific item on the page and spend less time describing what went wrong.
- No more vague emails. Bug report goes with technical information required to reproduce the issue.

== Frequently Asked Questions ==

= How can I integrate Widget into my website? =
1) Register on buglog and create a project in your profile.
2) After the project has been created, you will find the corresponding project id.
3) Install and activate the plugin.
4) Open Settings->Buglog, enter your project id and select the required access level.

= What languages does Buglog support? =
The feedback form supports the following languages:

- English
- German
- French
- Spanish
- Russian

= Can I change what Widget looks like? =
Yes, you can change the main color, position as well as choose dark or light appearance.
